# Account Management App

An account management application that allows users to register, log in, view their dashboard, manage their profile, and update personal information.

## Features

- User registration
- User login & authentication
- User dashboard
- View user profile
- Edit user profile
- Secure account management

## Usage
- Register a new account
- Log in with your credentials
- Access the dashboard
- View and edit your profile details
